## BMAD Method - 8-Tier Incremental Testing
**Business Marketing Acquisition Development** - 292 parameters tested incrementally.

### 8-Tier Structure (NEW - October 2025)
Testing organized by criticality, tested ONE tier at a time:
- **Tier 1**: Data Consistency (15 params) - 100% Required [BLOCKING]
- **Tier 2**: SEO Foundations (30 params) - 98% Required [CRITICAL]
- **Tier 3**: AI Search (25 params) - 98% Required [CRITICAL]
- **Tier 4**: Content Quality (40 params) - 98% Required [CRITICAL]
- **Tier 5**: Conversion/CRO (50 params) - 85% Required [HIGH]
- **Tier 6**: Psychology (45 params) - 98% Required [CRITICAL]
- **Tier 7**: Design & UX (60 params) - 85% Required [MEDIUM]
- **Tier 8**: Performance (27 params) - 70% Target [LOW]

**Testing Rule:** Test incrementally (little by little), not all at once. Each tier must pass before moving to next.

### Tools Location
- `tools/bmad-8tier-test.py` - NEW 8-tier incremental testing system
- `tools/bmad-visual-test.py` - Visual testing via Playwright MCP (full page screenshots)
- `tools/bmad-v2/` - Old modular testing system
- `tools/bmad-complete-test.py` - Legacy 277-param test

### Visual Testing (NEW)
**CRITICAL:** Always take full page screenshot after making changes to verify visual design.

Use Playwright MCP for visual verification:
1. Navigate to file:/// URL in browser
2. Take full page screenshot

## üèÜ GOLDEN TEMPLATE STRATEGY (Current Workflow)

**‚ö†Ô∏è CRITICAL: Follow this strategy to avoid bugs when creating/updating pages**

### Concept
Create ONE perfect "Golden Template" page first, test it thoroughly, THEN copy to other pages.

### Workflow Steps

**STEP 1: Create Golden Template** (Richmond Hill)
1. Optimize ONE page to 100% perfection
2. Test through ALL 8 BMAD tiers incrementally
3. Fix ALL issues tier by tier
4. Manually verify in browser (no bugs)
5. Get user approval before proceeding

**STEP 2: Test Copy on 2-3 Pages**
1. Copy Golden Template to 2-3 similar pages
2. Change ONLY: city name, title, meta, local content
3. Test Tier 1-4 (critical tiers) on each
4. User verifies 1-2 pages manually
5. If bugs found ‚Üí fix template ‚Üí re-copy

**STEP 3: Mass Copy in Batches**
1. Copy to 5 pages at a time (not all at once)
2. Test each batch with Tier 1-3 minimum
3. User spot-checks 1 random page per batch
4. If OK ‚Üí next batch, if bugs ‚Üí fix and re-run batch

**STEP 4: Brand Pages**
1. Repeat same process (create golden template first)
2. Test on 2-3 brands
3. Copy in batches of 3-4

### Why This Works
- No copying bugs to 30+ pages
- Each batch tested before next
- User verification at checkpoints
- Easy to rollback small batches
